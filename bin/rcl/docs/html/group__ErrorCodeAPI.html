Detailed Description

Unified error code table for RBQ.

Error code format (10 digits): D SSS TTT JJJ

D : Severity (1 = WARNING, 2 = FATAL) SSS : Subsystem TTT : Type JJJ : Index (Joint ID, port number, etc.)

Notes:

  • For Joint (SSS=001), JJJ is the joint number.
  • This class is for documentation and compile-time constants only.
